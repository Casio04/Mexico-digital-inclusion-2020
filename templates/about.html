About the project

The Population and Housing Census is the most important statistical project in the country. Given its universal nature, it is the source that, with the highest level of geographical disaggregation, provides information on the size, structure and distribution of the population in the national territory; and that allows to quantify the houses and their characteristics.

The intention of this project is to understand the level of digital inclusion that each state has reached so far through a visual and geographic approach. All the data and indicators presented were gathered through INEGI in the already mentioned Census

How?

Python

We used Python as main programming language to do the extract, cleaning and loading of the data.

Mongo DB

Mongo DB was the database that we used to storage the clean data.

Javascript

As main language for web visualization, we used JS in order to interact with DOM and set the content of our project.

D3

Used to create the custom interactive data visualizations.

Leaflet

To achieve the interactive map visualizations we used this open-source JS library.

Flask

Finally to access the data we perform an API using flask connecting with our database.
